- About -

This is a personal blog, where I share some extra svg files I have made. All files are in svg format, and are created using a text editor. Most icons are only 100 by 100 pixels and are in black and white. The source code of this SVG is valid 1.1. If you need help picking a color, there is a hexadecimal color reference linked here. We now accept donations/tips via PayPal. Visit us on Github, at publicsvg.github.io

- How To Download and Use -

First find an image that you whould like to use. On most devices you can simply click and the image will download on its own. However if this does not work for you simply right click the image and click save as. In case this does not work, try to open the file on a new window and view the source code. Then just copy and paste the text onto a text editor. Remember that svg is a software and you can change this software anytime you like.


You may also like to change or edit the color of your svg. One way to do this is to open your svg file with a text editor. Next find the part you whould like to change. This will have a # sign and three or six digits after it. If you need to there is a hexadecimal reference page here with several colors to choose from. Their are images that you can change the stroke that is the line's color, or the fill that is the background or fill-in color.
